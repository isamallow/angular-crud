> ## Sobre o Projeto
> O Angular CRUD é uma aplicação que cadastra produtos. O projeto foi no curso ministrado pela [Cod3r](https://www.cod3r.com.br/). Foi utilizado o Json-Server como back-end e Angular como front-end.
